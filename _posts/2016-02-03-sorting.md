#### 简单排序

选择排序

```c
void selection_sort(int arr[], int length) {
  int min;
  for (int i = 0; i < length; i++) {
    min = i;
    for (int j = i + 1; j < length; j++) {
      if (arr[j] < arr[min]) {
        min = j;
      }
    }
    swap(arr[i], arr[min]);
  }
}
```
